TSS Process: ENS Header -> Consignment -> SFD/EIDR -> GMR -> SDI (when required)
When TSS accepts a consignment and the goods are authorised for movement, it automatically issues an SFD reference and MRN, or holds non-controlled movements as EIDR. That customs reference is then included in the haulier's GMR. The supplementary declaration stage only follows on journeys where the simplified flow actually requires it.
