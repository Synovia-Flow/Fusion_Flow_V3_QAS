TSS References
ENS references import the ENS header only. TSS v2.9.5 reads consignments by their DEC reference, so Fusion cannot discover all child consignments from an ENS reference alone. Paste known DEC references to import consignments and their goods, and SUP references are saved as SDIs.
Expecting children? Import the parent ENS plus each known DEC reference. The ENS reference itself will not pull cargo rows from TSS.
